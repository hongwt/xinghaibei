# 校园AI指路小助手

这是一个为小学五年级AI创意竞赛设计的项目。它可以帮助家长在校门口快速找到孩子的班级，并自动规划路线。

## 功能特点
1.  **语音交互**：支持语音输入名字，语音播报结果。
2.  **智能查找**：自动查找学生班级，支持重名处理。
3.  **路径规划**：自动计算从校门口到班级的最短路线。
4.  **动态演示**：在地图上动态画出路线和小人移动动画。

## 文件结构
*   `main.py`: 主程序入口，包含界面和交互逻辑。
*   `data.py`: 存储学生名单和学校地图数据。
*   `logic.py`: 核心算法（查找学生、BFS路径规划）。
*   `voice.py`: 语音识别和合成模块。
*   `design_doc.md`: 详细设计文档。

## 如何运行

### 1. 安装依赖
请确保安装了 Python 3。
在终端中运行以下命令安装必要的库：

```bash
pip install pyttsx3 SpeechRecognition
```

*注意：如果需要使用麦克风进行实时语音识别，还需要安装 `pyaudio`。如果安装失败或无法使用，程序会自动切换到“键盘输入”模式。*

### 2. 启动程序
在终端中运行：

```bash
python main.py
```

## 使用说明
1.  点击“按住说话”按钮，说出学生名字（如“李华”）。
2.  或者在输入框中输入名字，点击“查询”。
3.  如果找到学生，地图上会显示红色路线，请跟随路线行走。
4.  如果有重名学生，系统会弹窗询问具体是哪个班级的。
